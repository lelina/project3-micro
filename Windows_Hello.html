Windows Hello :
Bạn chính là mật khẩu

Windows Hello chỉ mất chưa đến 2 giây để giúp bạn đăng nhập vào các thiết bị Windows, tức là nhanh hơn 3 lần so với khi dùng mật khẩu1.
Sử dụng máy ảnh để nhận diện khuôn mặt bạn hoặc thử dùng đầu đọc dấu vân tay8 – dù dùng cách nào thì Windows Hello cũng sẽ ngay lập tức nhận diện ra bạn.
Bạn luôn có thể dùng mã PIN để làm phương tiện dự phòng.

Đăng nhập theo cách của bạn

Việc kích hoạt Windows Hello sẽ bật tính năng đăng nhập bằng khuôn mặt hoặc dấu vân tay
Đăng nhập nhanh và bảo mật hơn vào máy tính xách tay, máy tính bảng, thiết bị, ứng dụng hoặc thậm chí là các trang web; hơn thế nữa, bạn còn có thể thực hiện mua tính năng mở rộng.

Thiết bị tương hợp mở khóa PC của bạn.
